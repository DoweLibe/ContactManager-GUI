# Contact Manager GUI

A simple Java Swing application for adding and searching contacts.  
Demonstrates event handling, input validation, and dynamic display of results using ArrayLists.

## Features
- Add contacts with name, email, and category
- Search contacts by name
- Display results in a scrollable text area
- Input validation and clean UI

## Skills Demonstrated
- Java Swing GUI development
- Event handling
- OOP (custom classes)
- ArrayList data structure
